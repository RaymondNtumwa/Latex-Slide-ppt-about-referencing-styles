# Descriptive Comparison between Referencing Styles

A LaTeX Beamer presentation comparing Harvard, IEEE, and APA referencing styles commonly used in academic writing.


This presentation provides a comprehensive comparison of three major referencing styles used in academic writing:
- **Harvard Style** - Common in UK universities and humanities
- **IEEE Style** - Technical fields, engineering, and computer science
- **APA Style** - Psychology, health sciences, and social sciences
